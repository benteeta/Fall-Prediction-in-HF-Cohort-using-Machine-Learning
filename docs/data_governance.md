# Data Governance
This repository does not contain patient-level healthcare data.

The original analyses were conducted using routinely collected healthcare data under institutional governance arrangements.
To protect patient confidentiality, the repository contains

- reproducible analysis scripts
- documentation
- synthetic demonstration data

Users wishing to reproduce the analyses with routinely collected healthcare data must obtain the appropriate approvals from the relevant data custodians.
No identifiable information has been included in this repository.
