# Study Design

## Study Type

Retrospective cohort study.

## Population

Adults with heart failure identified from routinely collected healthcare data.

## Outcome

Incident falls after heart failure diagnosis.

Falls were identified from

- Primary care records
- Hospital admission records

## Candidate Predictors

The following variables were considered.

### Demographics

- Age
- Sex
- Ethnicity
- Socioeconomic deprivation

### Lifestyle

- Smoking
- Alcohol consumption
- Body mass index

### Clinical

- Number of long-term conditions
- Frailty
- Previous falls
- Medication exposure
- Time since heart failure diagnosis

---

## Development Strategy

Patients were randomly divided into

- Development cohort (30%)
- Internal test cohort (20%)
- External validation cohort (50%)

using patient-level sampling.

---

## Machine Learning Models

Five supervised learning algorithms were compared.

The model with the highest discrimination was selected for external validation.

---

## Software

Analyses were performed using R.
